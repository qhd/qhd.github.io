Core Data 入门：iOS 数据持久化方案

Core Data 是 Apple 提供的数据持久化框架，它不仅仅是一个数据库，更是一个完整的对象图管理方案。本文将介绍 Core Data 的基本使用方法。

Core Data 是什么

Core Data 是一个对象关系映射（ORM）框架，主要功能包括：

  • 数据持久化（底层可以是 SQLite、XML 或内存）
  • 数据版本管理和迁移
  • 数据变更追踪
  • 数据验证
  • 延迟加载（Faulting）

核心组件

1. NSManagedObjectModel（模型）

定义数据结构，包括实体（Entity）、属性（Attribute）和关系（Relationship）。

2. NSPersistentStoreCoordinator（持久化存储协调器）

连接模型和实际存储，管理数据的读写。

3. NSManagedObjectContext（上下文）

管理对象的工作区，所有数据操作都在上下文中进行。

4. NSManagedObject（托管对象）

数据模型的实例，对应数据库中的一条记录。
