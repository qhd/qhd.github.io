Swift 4 Codable：告别手动 JSON 解析

Swift 4 引入了 Codable 协议，彻底改变了 Swift 中 JSON 解析的方式。不再需要第三方库，也不需要手动编写繁琐的解析代码。本文将全面介绍 Codable 的使用方法。

Codable 简介

Codable 是 Encodable 和 Decodable 的组合：

  • Encodable：将对象编码为外部表示（如 JSON）
  • Decodable：从外部表示解码为对象

总结

Codable 是 Swift 4 最实用的特性之一：

  1. 简洁：自动生成编解码代码
  2. 类型安全：编译期检查
  3. 灵活：支持自定义
  4. 性能好：原生实现，性能优秀

建议：

  • 简单场景直接使用自动合成
  • 复杂场景通过 CodingKeys 和自定义 init/encode 处理
  • 配合 keyDecodingStrategy 减少手动映射

Codable 已经成为 Swift JSON 处理的标准方案，强烈建议在新项目中使用。
